# Week-2--ML-Projects--AIMLTechies
ML Project tasks repo by group AIMLTechies
This project was built as part of our AIML internship Week 2 assignment. It covers Linear Regression implemented from scratch, and a classification model using KNN.
Used Diabetes csv dataset for KNN obtained from Kaggle.
Technologies Used:
- Python
- NumPy
- Pandas
- Matplotlib
Team Members: Beryl Zionah, Sankari Ganesh, Balaji, Kamalee, Prisha Kerin Mercy
How to Run the Project
1. Open Google Colab (colab.research.google.com)
2. Upload the .py file 
3. If running the KNN file, upload the diabetes.csv dataset into Colab first
4. Run all cells in order
